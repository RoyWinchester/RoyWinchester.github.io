# 机器学习

Arthur Samuel对机器学习的定义是

在没有明确设置的情况下，使计算机具有学习能力的研究领域。

Tom Mitchell提出的更新的定义是

计算机程序从经验E中学习，解决某一任务T，进行某一性能度量P，通过P测定在T上的表现因经验E而提高。

# 监督学习

给算法一个数据集，其中包含了正确的答案，算法的目的就是给出更多的正确答案。

主要是回归问题和分类问题。
