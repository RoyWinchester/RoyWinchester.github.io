# Remove Duplicates from Sorted Array

Given a sorted array nums, remove the duplicates in-place such that each element appear only once and return the new length.

Do not allocate extra space for another array, you must do this by modifying the input array in-place with O(1) extra memory.

Example 1: Given nums = [1,1,2], Your function should return length = 2, with the first two elements of nums being 1 and 2 respectively. It doesn't matter what you leave beyond the returned length.

# 我的解

我的思路是想生成一个新的数组，每次出现新的数组就加进去。

但是因为这个数组new_num = []是空的 ，所以不能直接加，就卡住了

看了一个思路类似的，是对原本的数组进行操作。贴上来吧。

def removeDuplicates(self, nums: List[int]) -> int:
    unique_count = 0
    if not nums: # 判断是否为空
        return unique_count
    current_num = None #当前数字
    current_idx = -1 #当前index
    for x in nums: #遍历
        if x != current_num: # 出现新数字
            new_idx = current_idx + 1 # index++
            nums[new_idx] = x #加到原本的数组里
            current_num = x  #当前数字更新
            current_idx = new_idx # 当前index更新
            unique_count += 1 # 长度++
    nums = nums[0:unique_count] #在长度切割
    return unique_count #返回长度

# 最优解

def removeDuplicates(self, nums: List[int]) -> int:
    if len(nums) == 0: # 0返回
        return 0
    i = 0
    for j in range(1,len(nums)):# 和上面那个逻辑一样 就是屁话少点
        if nums[j] != nums[i]:
            i += 1
            nums[i] = nums[j]
    return i+1
